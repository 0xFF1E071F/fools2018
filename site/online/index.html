TheZZAZZGlitch's April Fools Event 2018 - Online client

This is an HTML5-based online client for the event. It requires no configuration and can be played in most of the modern browsers.

There is only one thing that you have to provide on your own, and that's the English ROM of Pokémon Yellow (not included here, because of the copyright issues). Once you select the proper ROM, a JavaScript-based GBC emulator will start, the event save file will be automatically loaded, and an emulated link cable connection will be estabilished with the event servers. You will be logged in using the current account on the event site ().

Controls: Z for the A button, S or X for the B button, Enter for the Start button, Space for the Select button, arrow keys as the D-Pad.
