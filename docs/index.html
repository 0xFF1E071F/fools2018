TheZZAZZGlitch's April Fools Event 2018

Once upon a time, a new Pokémon region called Glitchland was discovered by Red, an adventurer from Kanto. Everyone was able to explore all of the secrets of this brand new world, using a specially crafted save file for Pok√©mon Yellow. All of the game's content was dynamically downloaded from the internet via a dedicated link cable protocol. The players would receive points and achievements for completing certain quests, and the progress was shown online on the event site, to let everyone battle for the top score.

On April 23rd, 2018, the event servers have been closed.

However, the idea itself has to live on. That's why source code for the client, the server, and the save file itself, has been released. It's for everyone who still wants to explore Glitchland or build something on top of it. Project page on GitHub.
